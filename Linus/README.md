Projet de biographie sur Linus Torvalds par Clément, Ahmad et Thomas Sladojevic.

Un environnement de bureau web interactif dédié à l'histoire et aux innovations de Linus Torvalds.

Comment afficher le site.

Le projet est entièrement statique. Vous n'avez besoin d'aucune installation complexe :

1. Téléchargez ce dossier.
2. Ouvrez le dossier du projet.
3. Double-cliquez sur le fichier `index.html`pour le lancer dans votre navigateur.

Fonctionnalités
- Interface de bureau type linux en lien avec le sujet de notre projet.
- Terminal Intégré** : Explorez les dossiers via des commandes (`ls`, `cd`) pour plus de lien avec le thème.

Projet développé en HTML, CSS et JavaScript.
